# Web Apps Collection

A collection of mini web applications built with **HTML**, **CSS**, and **JavaScript** to practice and showcase core web development skills.

## Projects

| #  | Project | Description | Demo |
|----|---------|-------------|------|
| 1  | [Calculator](JS-calculator/) | A functional calculator with keyboard support and basic arithmetic operations | [Open](JS-calculator/index.html) |
| 2  | [FD Calculator](FD-calculator/) | Fixed Deposit maturity calculator supporting simple and compound interest | [Open](FD-calculator/index.html) |
| 3  | [Password Generator](password-generator/) | Generate strong passwords with customizable length and character type options | [Open](password-generator/index.html) |
| 4  | [To-Do List](simple-todolist/) | Add, remove, and manage tasks with local storage persistence | [Open](simple-todolist/index.html) |
| 5  | [Code Editor](code-editor/) | A live HTML, CSS & JavaScript editor with real-time preview | [Open](code-editor/index.html) |

## Tech Stack

- HTML5
- CSS3 (Flexbox, Grid, CSS Variables, Transitions, Animations)
- Vanilla JavaScript (DOM Manipulation, Event Handling, Local Storage)

## Getting Started

1. **Clone the repository**
   ```bash
   git clone https://github.com/Agilkannan/web-applications.git
   ```
2. **Navigate into the repo**
   ```bash
   cd web-applications
   ```
3. **Open any project** - each folder is self-contained. Open the `index.html` file in your browser:
   ```bash
   # Example
   open JS-calculator/index.html
   ```

> No build tools or dependencies required. Just a browser!
